    Zoup

    Zoup is a python library inspired by the language Sage's group capabilities, utilizing groups as a main data type. This is still a work in progress, but down the line I would like to add a lot more capabilities so that it is comparable with Sage, and hopefully eventually get to graphic generation, similar to JulesPoo's Cayley Graphs and pretty things. I would also like to start getting it running like a full-scale opertaion with more utilization of github actions, version tracking and eventually pip installable. Group Theory is still a new topic for me and I am loving learning as I go!


    Streamlit OpenCV

    The idea bahind this project was originally to make a streamlit application connected to the spotify web api, putting a user's friend's information in a 3d-interactive plane to see how music tastes group and vary, using some dimension reduction algorithms. I was able to figure out how to get through the OAuth, as I realized you cannot access friends taste from a user's login. Thus, we move on! Newest idea is to make an openCV streamlit app, where you can input an image of a sodoku board and have a returned board. I think the first goal is jpeg upload or something, and then moving to real time camera input. This project is very much in the works, so be patient!
